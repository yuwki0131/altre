# Tauri GUI ドキュメント更新

## タスク概要
Tauri GUI 追加に伴う README / INSTALL / manuals / AI_DEBUG_GUIDE 等のドキュメント更新を行う。

## 目的
- 新しいビルド手順、依存関係、トラブルシューティングを利用者・開発者向けに整理する。
- GUI/TUI の機能差分や操作方法をわかりやすく記述する。
- ログ出力設定や検証手順をガイド化する。

## 成果物
- `README.md` に Tauri GUI セットアップ方法（React/Tauri ビルド手順、依存リスト）を追記。
- `INSTALL.md` や `manuals/` に GUI 版の操作方法、既知の制限、NixOS 向け注意点を追加。
- `AI_DEBUG_GUIDE.md` に GUI デバッグフロー（ログ、テストコマンド）を反映。
- 変更内容を `log.md` および `CHANGELOG.md` に追記。
